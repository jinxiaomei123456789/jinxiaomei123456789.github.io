系统信息

  windows系统加固之前，需要查看windows的系统信息，通常需要查看的信息如下：
  1.查看系统版本，使用命令ver

  2.查看SP版本，SP是指系统的补丁包,使用命令wmic os get ServicePackMajorVersion

  3.查看Hotfix,Hotfix是微软公司研发的一个程序，针对某一个具体的系统漏洞或安全问题而发布的专门解决该漏洞或安全问题，通常称为修补程序。使用命令wmic qfe get hotfixid

  4.查看windows的主机名hostname,使用命令hostname

  5.查看网络配置，使用命令ipconfig/all

  6.查看路由表，使用命令route print

  7.查看开放端口，使用命令netstat -ano

补丁管理

  1.补丁安装的目的：修补漏洞
  2.注意事项：打补丁前做好业务或系统备份，回退时恢复备份。补丁安装后可能影响业务系统的稳定性。

账号口令

账户

  操作目的：减少系统无用账户，降低风险。
  检查方法：开始->运行->compmgmt.msc(计算机管理)->本地用户和组，查看是否有不用的账号，系统账户所属组是否正确，以及guest账户是否锁定。

口令策略

  操作目的：增强口令的复杂度及锁定策略等，降低被暴力破解的可能性。
  检查方法：开始->运行->secpol.msc(本地安全策略)->安全设置
  加固方法：账户设置->密码策略，要求如下：
    密码必须符合复杂性要求：启用
    密码长度最小值：8个字符
    密码最短使用期限：0天
    密码最长使用期限：90天
    强制密码历史：5个记住密码（更改的密码不能和前5次相同）
